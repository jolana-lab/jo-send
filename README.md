# Jo Send

It's an Application that allows users **send cryto coins/token to friends on chat applications**.

## Supported cryto coins/token

- [x] SOL

## Supported chat applications

- [x] Slack
- [ ] Discord

## Development

1. open development proxy

   ```command
   ngrok http 3000
   ```

2. run nest dev server

   ```command
   npm run start:dev
   ```
